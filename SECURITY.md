# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 1.x     | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

NailSnake is a terminal game that does not handle untrusted input or network
data, so the attack surface is minimal. However, if you discover a security
vulnerability, please report it privately.

**Do not open a public GitHub issue.** Instead, email the maintainer directly:

**voltsparx@gmail.com**

You should receive a response within 48 hours. If you do not, please follow up.

### What to include

- Description of the vulnerability
- Steps to reproduce
- Affected versions
- Any potential impact

### Disclosure

We will acknowledge receipt within 48 hours and aim to release a fix within
14 days. Once a fix is released, we will publicly disclose the vulnerability
after a reasonable period for users to update.
